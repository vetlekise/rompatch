# rompatch

A Go command-line utility for patching video game ROMs.

## Features
- Patching formats:
    - IPS

## Installation

**Using Go:**

```sh
go install github.com/vetlekise/rompatch@latest
```

**Pre-built binaries:**

Download the latest release for your platform from the [GitHub Releases](https://github.com/vetlekise/rompatch/releases) page.
